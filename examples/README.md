# Examples

This directory contains practical examples demonstrating how to use the GitHub Achievement Unlocker tool.

## Available Examples

### 1. [Basic Usage](./basic-usage.md)
Learn the fundamental commands and how to get started with the tool.

### 2. [Achievement Tracking](./achievement-tracking.md)
Discover how to track your progress towards unlocking GitHub achievements.

### 3. [Custom Scripts](./custom-scripts/)
Integration examples and custom automation scripts.

## Quick Start

```bash
# Clone the repository
git clone https://github.com/galafis/github-achievement-unlocker.git
cd github-achievement-unlocker

# Install dependencies
npm install

# Try the basic examples
npm start list
npm start check octocat
npm start template
```
